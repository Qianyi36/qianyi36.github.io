HTML

Intro

HTML is the standard language used for creating websites, it stands for Hyper Text Markup Language

The current version of HTML is HTML 5, which is what we will be using.

Setup

IDE

You need a place to start coding HTMl. My prefered IDE is Visual Studion Code(Which I used to code this site).

Setting up a file

Make a folder with the name of your choice, this is where your code will go.

In visual studio, click file, then open folder, select the folder you just created.

Create a new file in the folder called index.html, this is the homepage for your website

Syntax

HTML has relatively simple syntax, it is quite easy to understand.
