HTML

Intro

HTML is the standard language used for creating websites, it stands for Hyper Text Markup Language

The current version of HTML is HTML 5, which is what we will be using.

Setup

IDE

You need a place to start coding HTMl. My prefered IDE is Visual Studio Code(Which I used to code this site).

Setting up a file

When creating files and folders, do NOT capitalize anything, capitals will mess you up when you go to link multiple files together.

Make a folder with the name of your choice, this is where your code will go.

In visual studio, click file, then open folder, select the folder you just created.

Create a new file in the folder called index.html, this is the homepage for your website

Syntax

HTML has relatively simple syntax, it is quite easy to understand.

All HTML items are written as tags, with a word of letter(s) inside of angled brackets.

Most tags will begin with a <tag> and end with </tag>, however, tags that do not need anything inside of them are written as <tag/>, or just <tag>.

For example, the <p>Hello</p> tag will write Hello on your website

Code

When you create your website, you want to start with the doctype declaration, it is written as <!DOCTYPE html>.

It tells the browser the version of html that your are using, and helps the site load properly.

Then add the <html> tag, every thing should begin and end with the html tag

Comments

Comments are written as <!--Comment-->, these are to make your code readable, and to explain what it is doing.
