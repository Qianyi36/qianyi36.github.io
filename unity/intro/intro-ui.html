Unity

This page is a basic overview of the Unity User Interface

The User Interface

The Unity user interface has 4 major parts, which are the hierarchy, asset viewer, the game view, and the inspector.

I don't actually know what the "asset viewer" and "game view" are called, as they are not labeled with their names, so I made up a name that makes sense.

Hierarchy

The hierarchy shows all of game objects currently in the game, when starting up the project, a camera will be in the hierarchy(and a directional light, if 3d was selected).

Game objects can be created by dragging sprites, or prefabs into onto the hierarchy, or by center clicking.

Game View

The game view is where you will edit and view the game, when you create the game, there will be a tab labeled scene, and one labeled game. Scene view allows you to edit and move game objects, and view the game from different angles, game view shows what the player would see, the view encompassed by the range of the camera(Shown by a white rectangle in scene view).

A problem that happens to me occasionally, is the absence of a tab when I enter the game. If this happens to you, center click on the tab that is there, and click "add tab" then add what you need.

Asset viewer

The asset viewer is Unity's file explorer, it lets you organize all of your files into folders. Game objects are stored in the hierarchy, files such as sprites, scripts, and prefabs that are not gameobjects are stored in the asset viewer.

center clicking in the asset viewr allows you to create a variety of different objects, or import something from your computer.

Inspector

The inspector lets you view the components and details of game objects, clicking on a game object in the hierarchy, or clicking it in scene view, pulls up the game object in the inspector.

Clicking on the square game object, visible in the game view, allows me to view and edit its components in the inspector.
