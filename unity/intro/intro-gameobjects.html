A guide to Gameobjects

The content of this page will mostly be geared toward 2d games, but if you are making 3D games, learn this fist, then add on by learning how to do 3D game develoment

Gameobjects

This page will give a brief overview of what gameobjects are, and how to manipulate them in your game.

What are Gameobjects

A game object is an object that is physically present in your game, anything that shows up in your hierarchy is a gameobject.

Creating Gameobjects

Game objects can be created by dragging sprites and prefabs from your assets to the hierarchy, or by right clicking on the hierarchy

Types of Gameobjects

  • Empty(can be used to store scripts or perform tasks without being seen by the player)
  • 3D object (includes such as cubes of spheres)
  • 2D object (A sprite has to be applied)
  • Tilemap (A grid system that allows for easier tile based developement.)
  • Effects (Particles, lines, trails)
  • Audio source
  • lights
  • Video player
  • Game ui

Editing Game objects

In scene view, you can drag gameojects around to approximate positions, if you want to get more precise, you can edit the location through the inspector.

In the inspector, other charictaristics of game objects can be changed as well, such as rotation and the scale of the object.
