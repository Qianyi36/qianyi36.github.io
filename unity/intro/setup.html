Unity

This page explains how to setup untiy

Installing Untiy

Download Unity to your computer by clicking the Personal option, clicking get started, and running the installer. Make sure to install Unity Hub.

I use Unity 2019, but later versions should work as well.

Other versions of Unity can be installed through the installs tab on Unity Hub.

Making a game

Open unity hub, which should have been installed along with unity

Click new, and select the game type you want to make
